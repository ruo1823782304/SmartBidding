# 前端功能与后端接口对照表

**原则**：后端实现不得减少前端已有任何功能，接口与字段以《接口表与字段说明》为准。

---

## 一、认证与用户（前端：登录页、顶栏用户、编辑资料）

| 前端功能 | 方法 | 路径 | 必实现 |
|----------|------|------|--------|
| 登录 | POST | /api/auth/login | ✓ |
| 登出 | POST | /api/auth/logout | ✓ |
| 获取当前用户 | GET | /api/user/me | ✓ |
| 编辑资料（姓名、联系方式、改密） | PUT | /api/user/profile | ✓ |

---
